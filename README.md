## About

This list focuses exclusively on testers persforming **physical hardware testing, biological testing, and industrial certification services**. We cover:

- **Hydrogen & Energy:** High-pressure (700 bar), cryogenic, and fuel cell component validation.
- **Biological Testers:** Pharmaceutical, biocompatibility, preclinical, biosafety.
- **Laboratory Testers:** EMC, HPLC, GC-MS, tensile strength, metallurgy.
- **Industrial Testers:** Environmental chambers, HALT/HASS, salt spray, IP ratings.
- **Certification:** UL, CE, ISO 17025, A2LA accreditation.

**Explicitly Excluded:** Software QA, unit testing frameworks, code coverage tools, and digital security testing.

**Why This Matters:** Finding specialized testing labs is notoriously difficult. Engineers waste days searching for labs certified for specific standards (e.g., **ISO 19880-3**, **SAE J2601**) or capabilities (cryogenic hydrogen, -40¬∞C to +85¬∞C thermal cycling). This list aggregates the best providers globally.

* **[CSA Group](https://www.csagroup.org)** - *Hydrogen Components*
    * North American leader in testing hydrogen dispensing systems, valves, and hoses to ANSI/CSA HGV standards.
* **[T√úV S√úD](https://www.tuvsud.com/en/industries/energy/hydrogen)** - *H2 Readiness*
    * Offers comprehensive "H2 Readiness" certification for materials, piping, and power plants, plus electrolyzer safety testing.
* **[Element Materials Technology](https://www.element.com/energy/hydrogen)** - *Materials Compatibility*

    * Specialized testing for maritime and industrial hydrogen fuel cell power systems.

## Contributing

Contributions are welcome! Please ensure your submission meets the following criteria:

### Submission Guidelines

1.  **Physical Lab Address Required:** Provider must operate at least one physical testing facility. No pure consulting firms or software tools.
2.  **Active Operations:** Lab must be currently operational.
3.  **Niche or Specialized:** Prefer labs with specific certifications (ISO 17025, A2LA), unique capabilities (cryogenic, high pressure), or industry specialization.
4.  **Format:** `* **[Lab Name](URL)** - *Brief description including key capabilities.*`
5.  **Pull Request:** Fork this repo, add your entry under the appropriate category, and submit a PR.

### Quality Standards

* **No Duplicates:** Search the list first.
* **No Spam:** One entry per provider.
* **Accurate Links:** URL must resolve to provider's official website.
